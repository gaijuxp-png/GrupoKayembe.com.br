# Grupo Kayembe — Site Institucional

Site institucional do **Grupo Kayembe**, escritório de contabilidade fundado em **24 de agosto de 2022**, em Belém/PA.

Site one-page, responsivo, sem dependências externas de JavaScript. Construído em HTML + CSS puro com tipografia do Google Fonts (Fraunces + Inter).

## Estrutura

`
kayembe-site/
├── index.html        # Página única
├── style.css         # Estilos (paleta branco/vermelho/preto)
├── img/              # Imagens (logo, fotos, galeria)
└── README.md
`

## Stack

- HTML5 semântico
- CSS3 com variáveis customizadas (--vermelho, --preto, --branco)
- Google Fonts: Fraunces (títulos) + Inter (corpo)
- 100% responsivo (mobile-first com breakpoints em 860px e 480px)

## Seções

1. **Header** — Logo do grupo + navegação
2. **Hero** — Foto destaque do casal fundador + CTA WhatsApp/Instagram
3. **Sobre** — História do escritório (fundado em 24/08/2022)
4. **Serviços** — 6 áreas: Abertura, Fiscal, Folha, Consultiva, Planejamento, Regularização
5. **Ecossistema** — Empresas do grupo:
   - Grupo Kayembe (holding)
   - Kayembe Contabilidade
   - Kayembe & Moreira Advogados
   - Kayembe Imóveis
   - Kayembe Service
   - Kayembe Ucipu
6. **Galeria** — Bastidores (Instagram @grupo.kayembe)
7. **CTA** — Botão WhatsApp + link Instagram
8. **Footer** — Endereço, contato, créditos
